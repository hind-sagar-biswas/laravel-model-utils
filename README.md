A lightweight Laravel package for Model traits and utilities

## Installation

You can install the package via composer:

```bash
composer require hindbiswas/laravel-model-utils
```


## Usage

```php
$modelUtils = new HindBiswas\ModelUtils();
```

## Testing

```bash
composer test
```
